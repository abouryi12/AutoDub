# =============================================================================
# TROUBLESHOOTING
# =============================================================================
# Common issues and solutions:
#
# 1. PyTorch CUDA errors:
#    - Install CUDA version matching your system: 
#      pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#
# 2. FFmpeg not found:
#    - Install FFmpeg and add to system PATH
#    - Verify with: ffmpeg -version
#
# 3. TTS voice not available:
#    - Check Edge TTS version compatibility
#    - Some voices may be region-restricted
#
# 4. YouTube download failures:
#    - Update yt-dlp: pip install --upgrade yt-dlp
#    - Use authentication cookies for private videos
#
